# Open Port Check Script
This function fsockopen() is used in the Open Port Check Tool for our private blog. You can check whether a certain port is open on a specified IP address or hostname. Before reading take note that most FREE web hosting providers disable fsockopen() function as a security measure so first check whether the function can be used.
